# `@mkhotsevich/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
yarn add --dev @mkhotsevich/prettier-config
```

or

```bash
npm install -D @mkhotsevich/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@mkhotsevich/prettier-config"
}
```
